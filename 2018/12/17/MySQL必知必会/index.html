MySQL必知必会

发表于 2018-12-17 | 分类于 MySQL

SQL语句

查看数据库和表信息

> SHOW DATABASES; //查看所有数据库

> SHOW CREATE DATABASE; //查看特定数据库创建语句。

检索

查询所有字段

SELECT * FROM TABLE;

查询不重复行的特定字段

SELECT DISTINCT id FROM TABLE;

LIMIT限制返回数量

SELECT FROM TABLE LIMIT 5;
SELECT FROM TABLE LIMIT 5,5; //从第5行开始的5行。

ORDER BY排序

SELECT FROM TABLE ORDER BY COLUMNS;
SELECT FROM TABLE ORDER BY COLUMN1, COLUMN2;
SELECT * FROM TABLE ORDER BY COLUMN1 DESC, COLUMN2 DESC;

WHERE 条件查询

!= 和 <> 都表示不等于
BETWEEN 查询间隔 ： SELECT * FROM TABLE WHERE COLUMN BETWEEN a AND b;
空值检查NULL： IS NULL 判断NULL
AMD 和 OR 多条件查询， 注意查询次序
IN 表示范围匹配
使用 NOT 对IN、BETWEEN、EXECTS取反

通配符检索

% :表示任意字符出现任意多次
SELECT FROM TABLE WHERE name LIKE ‘wang%’;
_ :表示任意字符出现一次
SELECT FROM TABLE WHERE name LIKE ‘wan_%’;

REGEXP 表示使用正则表达式检索 . *

SELECT FROM TABLE WHERE name REGEXP ‘.000’;
使用 \\ 进行特殊字符的转义
定位符 ^ 字符串开头 $ 字符串结尾
. 匹配一个
\ 匹配0个或多个
+ 匹配1个或多个
? 匹配0个或1个
